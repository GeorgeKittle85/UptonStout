How this works: Upload a clear, front-facing photo. Two pipelines run in parallel — MediaPipe Face Mesh (468 landmarks → 7 classical methods) and a Convolutional Neural Network via face-api.js (ResNet-based → 128-D embedding). Results from both are combined into a final weighted score.
Neural Network: A ResNet-34-inspired CNN has been pretrained on thousands of faces to produce a 128-dimensional embedding vector where faces of the same person cluster close together in embedding space. This is the same principle used by FaceNet, ArcFace, and most modern face recognition systems.
